## Design Stages

# 1.
Basic proof of concept implemented in HTML, JaveSript and TypeScript utilizing BRC 100 SDK for basic wallet management.
Functionality leveraging P token protocol integrated at the app level.

# 2.
Refactoring to incorporate P token protocol into a development branch of BRC 100 SDK and remove from app.

# 3.
Adding prototype app to Metanet apps for wider testing by the BSV community.

# 4. 
Release of SVphone as a browser extension for all popular browsers.

# 5.
Development of a standalone app for Windows, Mac and Linux.

# 6.
Development of a mobile app for submission to Google Play and Apple App Store.

# 5.
Ivestigating the feasibility of providing an upgrade path for old phones to use a KDE or similar mobile OS incorporating Metanet.
